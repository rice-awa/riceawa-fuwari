B站上有许多优质的学习资源和优秀的二次创作视频。有时候为了方便离线观看或者存档，我们需要将视频下载到本地。

今天推荐三种不同维度的下载方法，涵盖了**在线网站、浏览器插件以及专业客户端**，总有一种适合你！

---

### 方法一：最便捷——SaveAny 在线下载（无需安装）

如果你不想安装任何软件，只是偶尔下载一两个视频，那么使用在线解析网站是最简单的方案。

*   **工具名称：** SaveAny
*   **访问地址：** [https://saveany.cn/](https://saveany.cn/)
*   **操作步骤：**
    1.  打开 B 站，复制你想要下载的视频页面 URL（链接）。
    2.  访问 **SaveAny** 官网。
    3.  将链接粘贴到搜索框中，点击解析按钮。
    4.  解析完成后，点击链接下载即可。
*   **优点：** 跨平台（手机、电脑都能用），无需安装，即下即走。

---

### 方法二：最原生——Edge 浏览器下载助手（一键下载）

如果你经常在电脑端使用浏览器看 B 站，安装一个插件可以实现“边看边下”，操作体验非常原生。

*   **工具名称：** Bilibili 哔哩哔哩下载助手
*   **获取地址：** [Microsoft Edge 拓展商店](https://microsoftedge.microsoft.com/addons/detail/bilibili%E5%93%94%E5%93%A9%E5%93%94%E5%93%A9%E4%B8%8B%E8%BD%BD%E5%8A%A9%E6%89%8B/cagicamgdlbdmonbclkpgiabbldodgae)
*   **操作步骤：**
    1.  点击上方链接，将插件添加到你的 Edge 浏览器（Chrome 浏览器通常也可通用）。
    2.  打开任意一个 B 站视频播放页面。
    3.  你会发现在视频页面右下方或侧边出现了一个“下载视频”按钮。
    4.  点击“合并下载”按钮，即可进行下载。
*   **优点：** 与网页集成度高，方便快捷。

---

### 方法三：最强大——Downkyi（哔哩下载姬）

如果你需要批量下载整个收藏夹、整个 Up 主的视频，或者需要下载 4K 超高清画质，那么这款开源的桌面客户端是你的首选。

*   **工具名称：** Downkyi (哔哩下载姬)
*   **获取地址：** [GitHub Releases](https://github.com/yaobiao131/downkyicore/releases)
*   **操作步骤：**
    1.  在 GitHub 发布页下载最新的压缩包并解压运行。
    2.  登录你的 B 站账号（扫码登录后可下载已购内容或更高清晰度）。
    3.  将视频链接、直播间链接或个人主页链接粘贴进输入框。
    4.  设置好下载路径，勾选需要的文件，开启批量下载。
*   **优点：** 支持**批量下载**、支持断点续传、支持 4K/HDR/杜比视界（需大会员账号）、支持解析课程。

---

### 总结：我该选哪一个？

- **偶尔下载、不想折腾：** 选 **方法一 (SaveAny)**。
- **追求操作方便、随看随下：** 选 **方法二 (浏览器插件)**。
- **专业需求、批量下载高清资源：** 选 **方法三 (Downkyi)**。
